# Roadmap

Reticula is in **early alpha**. This roadmap shows direction, not commitments —
**no dates are promised**, and priorities may change.

Checked items are implemented today; unchecked items are planned.

## 0.1.0-alpha — Foundation (current)

- [x] Flutter app structure
- [x] Real-size page model (millimetre-based)
- [x] Multiple paper sizes (A3–A6, Letter, Legal, 10 × 15, 13 × 18)
- [x] Configurable photo grids (1–12 cells), portrait / landscape
- [x] Image import
- [x] Manual crop / position / zoom adjustment
- [x] PDF export
- [x] PNG and JPEG export at 300 DPI
- [x] Localization (7 languages)

## 0.2.0-alpha — Project usability

- [ ] Save / load `.reticula` projects
- [ ] UI polish
- [ ] More layout presets
- [ ] Improved image handling
- [ ] Drag-and-drop image import
- [ ] Basic print guidance / system print dialog

## 0.3.0-alpha — Cross-platform desktop

- [x] Windows build (CI, experimental)
- [x] Linux build (CI, experimental)
- [x] macOS build
- [x] GitHub Releases artifacts
- [ ] Broader real-world testing on each platform

## Future

- Android support
- iOS support
- More paper and photo presets
- Contact sheets
- Custom / freeform layouts
- Template sharing
- More translations and accessibility improvements

(Internationalization has already started — additional languages are welcome.)
