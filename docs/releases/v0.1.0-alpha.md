# Reticula 0.1.0-alpha

This is the first public alpha of Reticula, a free and open source photo layout
tool focused on real print sheets.

Reticula aims to help people arrange photos on real paper formats, with control
over size, margins, crop, and framing.

## Status

This is an early alpha. Expect missing features, rough edges, and rapid changes.

## Current focus

- Real-size page layout (A4 and other paper sizes)
- Configurable photo grids (for example, 2 A5 photos on an A4 sheet)
- Marginless layout
- Manual photo framing (pan / zoom / crop)
- PDF export
- PNG and JPEG export at 300 DPI
- A seven-language interface
